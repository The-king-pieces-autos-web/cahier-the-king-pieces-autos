CORRECTION SESSION + UTILISATEURS

1) La connexion reste enregistrée dans localStorage avec id + identifiant + rôle.
2) Après actualisation, le site recharge automatiquement l’utilisateur connecté.
3) Dans Utilisateurs, l’admin peut supprimer les anciens utilisateurs.
4) Bouton ajouté : Supprimer anciens utilisateurs, garde uniquement le compte admin connecté.

Après déploiement :
- Connecte-toi en admin
- Va dans Utilisateurs
- Clique Supprimer anciens utilisateurs
- Crée tes vrais comptes salariés
