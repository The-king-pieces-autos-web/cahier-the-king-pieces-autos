SAUVEGARDE AUTOMATIQUE AJOUTÉE

Cette version ne touche pas aux données déjà enregistrées dans Supabase.
Elle modifie uniquement le code du site.

Fonctionnement :
- Une sauvegarde automatique est créée toutes les 1 heure.
- Elle apparaît dans le menu "Sauvegardes jour".
- Chaque sauvegarde automatique contient :
  - utilisateurs
  - fiches cahier salarié
  - devis
  - résumé par salarié
- Elle ne remplace pas les fiches et devis existants.
- Elle n'efface rien.
- Elle évite les doublons sur la même heure.
- Elle conserve jusqu'à 80 sauvegardes automatiques.

Bouton ajouté :
- "Créer une sauvegarde automatique maintenant"
