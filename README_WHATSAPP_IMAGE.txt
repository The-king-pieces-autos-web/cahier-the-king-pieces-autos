CORRECTION WHATSAPP + IMAGES

WhatsApp :
- Le message est structuré comme un devis propre :
  - Bonjour + nom client
  - numéro devis
  - client
  - véhicule
  - origine demande
  - détail pièces
  - disponibilité
  - total HT / TVA / TTC
  - remerciement

Images :
- Les images ajoutées ou collées s'affichent entièrement.
- Elles ne sont plus coupées.
- Le bouton de collage direct reste disponible.
