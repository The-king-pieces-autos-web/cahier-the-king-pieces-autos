ENVOI WHATSAPP COMME IMAGE

Important :
WhatsApp Web ne permet pas à un site web d'attacher automatiquement une image sans API WhatsApp Business officielle.

Cette version fait donc :
1. Génère le devis en image PNG.
2. Télécharge automatiquement l'image.
3. Ouvre WhatsApp avec le message prêt.
4. Tu joins l'image téléchargée dans WhatsApp et tu envoies.

Boutons ajoutés :
- Télécharger image
- WhatsApp : génère l'image puis ouvre WhatsApp.
