COLLER UNE IMAGE DIRECTEMENT

Dans Cahier salarié :
1. Copie une image depuis internet ou un logiciel.
2. Clique dans le cadre "Coller image" de la proposition.
3. Fais Ctrl+V.
4. L'image s'ajoute directement à la référence.

Le bouton "Ajouter fichier" reste disponible si tu veux choisir une image depuis ton PC.
